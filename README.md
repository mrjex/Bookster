## Getting Started

1. **Prerequisites**
   - Node.js (v16+)
   - MongoDB
   - npm or yarn

2. **Installation**
   ```bash
   # Clone repository
   git clone https://github.com/yourusername/Bookster

   # Run the client
   cd Bookster/client && npm i && npm run serve

   # Run the server
   cd Bookster/server && npm i && npm run dev
   ```
